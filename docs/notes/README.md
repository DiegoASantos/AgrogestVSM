# Notas temporales

Esta carpeta puede recibir bitácoras, borradores y notas creadas mediante
Obsidian u OpenGem.

Las notas:

- no son fuente oficial;
- deben enlazar al tema relacionado;
- no pueden contradecir documentos activos sin señalarlo;
- deben revisarse y promoverse o eliminarse.

Una nota relevante se promueve a:

- ADR, si contiene una decisión;
- spec, si propone un cambio crítico;
- runbook, si describe un procedimiento;
- arquitectura o dominio, si describe el estado vigente;
- registro de riesgos, si identifica exposición activa.
