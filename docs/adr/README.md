# Registros de decisiones arquitectónicas

Los ADR registran decisiones importantes y su contexto. Una decisión aceptada
no se reescribe para ocultar el historial. Si cambia, se crea un nuevo ADR y el
anterior se marca como `superseded`.

## Numeración

```text
001-titulo-decision.md
002-otra-decision.md
```

## Estados

- `proposed`
- `accepted`
- `superseded`
- `rejected`
